## Tech Stack

## Run Locally

Clone the project

```bash
  git clone https://github.com/SyedaMahamFahim/Brm-Calculator-React.git
```

Go to the project directory

```bash
  cd brm-calculator-react

```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
